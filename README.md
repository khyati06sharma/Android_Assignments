# Android Assignments

This repository contains Android Studio assignment projects developed using Kotlin and Android Studio.

## 📱 Technologies Used
- java
- Android Studio


## 📂 Project Structure
- app/
- gradle/
- build.gradle.kts
- settings.gradle.kts

## 🚀 How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run the application on Emulator or Physical Device

## 📌 Note
These projects are created for learning and academic purposes.
